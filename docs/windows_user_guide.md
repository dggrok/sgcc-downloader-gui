# SGCCDownloader 用户使用说明

## 1. 适用对象
- 需要在 Windows 上检索国网 ECP 公告并批量下载 PDF 附件的业务或运营人员

## 2. 启动方式
- 解压交付包
- 双击 `SGCCDownloader.exe`
- 一般情况下不需要安装 Python

## 3. 使用流程
### 第一步：选择机构
- 在左侧机构树中逐级展开选择机构
- 或在上方输入机构关键字后点击“搜索机构”
- 选择后，界面会显示当前机构名称和机构 ID

### 第二步：设置查询条件
- 输入公告关键词，例如“物资”
- 设置起始页、每页数量、最大页数
- 设置下载目录
- 如需区分每次任务，勾选“为本次任务创建子目录”

### 第三步：预览附件
- 点击“查询公告”
- 程序会查询公告并展示附件列表
- 默认所有结果已勾选

### 第四步：下载附件
- 可取消不需要的勾选项
- 点击“下载选中”
- 等待进度条完成
- 点击“打开目录”查看文件

## 4. 下载目录说明
- 支持手工输入本地目录
- 支持点击“浏览...”选择目录
- 支持点击“恢复默认”回到默认下载目录
- 程序会记住上一次使用的目录

## 5. 常见问题
### 没有查到结果
- 检查机构是否选择正确
- 检查关键词是否过于严格
- 适当增加页数范围

### 下载失败
- 检查网络是否可访问目标网站
- 检查下载目录是否有写入权限
- 重新尝试，必要时缩小查询范围

### 想换保存位置
- 直接修改“下载目录”输入框
- 或点击“浏览...”重新选择目录

## 6. 使用建议
- 首次使用先用较小页数试跑
- 下载量大时，建议分批查询和下载
- 如遇异常，可将日志内容反馈给维护人员排查
